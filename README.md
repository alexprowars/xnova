## Description

Classic Ogame clone writen on php/laravel and vue.js/NuxtJS

## Frameworks

Laravel (**https://laravel.com**) \
Inertia (**https://inertiajs.com**) \
Vue (**https://vuejs.org**)

## System requirements:
- PHP 8.5 and higher
- MySQL 8.0 and higher
- NodeJS 24 and higher
- Composer

## Run in docker
`composer install`\
`cd docker`\
`docker-compose build --no-cache`\
`docker-compose up -d`\
`docker exec xnova composer setup`
